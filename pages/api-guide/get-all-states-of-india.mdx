## Get All States Of India



Technical details regarding the usage of the pincode endpoint can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Route Schema (URL)


```
http://localhost:3000/state

```

## Request Samples

<Tabs items={["JavaScript", "Python"]}>
  <Tab>
    <>
        ```js copy 
        import axios from "axios";

        // Using the example and looking at the results.
        const url = "http://localhost:3000/state";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
