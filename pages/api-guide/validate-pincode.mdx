# Validate Pincode

Technical details regarding the usage of the pincode validation endpoint can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Route Schema (URL)


```
http://pinpoint.devildesk.wip.la/pincode/{pincode}/validate

```


## Path Parameters

| Parameters |   Type    |                                     Description                                          | Required? | Default |
|:----------:|:-------: |:----------------------------------------------------------------------------: |:--------: |:------:|
| pincode    |  Integer  | The pincode path parameter; i.e. the pincode for which info you are looking for |     Yes    |    ""   |


## Request Samples

<Tabs items={["JavaScript", "Python"]}>
  <Tab>
    <>
        ```js copy 
        import axios from "axios";

        // Using the example pincode 411057, and looking at the results.
        const url = "http://pinpoint.devildesk.wip.la/pincode/411057/validate";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>

        data = response.json()
