# Get Pincode Info

Technical details regarding the usage of the pincode info endpoint can be found below. Example code is provided for both JavaScript and Python, along with a response schema.

## Route Schema (URL)


```
https://tasteless-lari-person-project-41fd5ca1.koyeb.app/pincode/{pincode}/info

```


## Path Parameters

| Parameters |   Type    |                                     Description                                          | Required? | Default |
|:----------:|:-------: |:----------------------------------------------------------------------------: |:--------: |:------:|
| pincode    |  Integer  | The pincode path parameter; i.e. the pincode for which info you are looking for |     Yes    |    ""   |


## Request Samples

<Tabs items={["JavaScript", "Python"]}>
  <Tab>
    <>
        ```js copy 
        import axios from "axios";

        // Using the example pincode 411057, and looking at the results.
        const url = "https://tasteless-lari-person-project-41fd5ca1.koyeb.app/pincode/411057/info";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>
