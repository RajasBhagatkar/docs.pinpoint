# Get All Districts/Cities Of State

Technical details regarding the usage of the state endpoint can be found below. This endpoing provides all the districts which are found in provided state. Example code is provided for both JavaScript and Python, along with a response schema.

## Route Schema (URL)


```
https://pinpoint.devildesk.wip.la/state/district/{state_name}

```


## Path Parameters

| Parameters |   Type    |                                     Description                                          | Required? | Default |
|:----------:|:-------: |:----------------------------------------------------------------------------: |:--------: |:------:|
| state_name    |  String  | The state_name path parameter; i.e. the state for which cities / districts you are looking for |     Yes    |    ""   |


## Request Samples

<Tabs items={["JavaScript", "Python"]}>
  <Tab>
    <>
        ```js copy 
        import axios from "axios";

        // Using the example state_name is maharashtra, and looking at the results.
        const url = "https://pinpoint.devildesk.wip.la/state/district/maharashtra";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>
  <Tab>
    <>
        ```python copy 
        import requests

        # Using the example state_name is maharashtra, and looking at the results.
        url = "https://pinpoint.devildesk.wip.la/state/district/maharashtra";
        response = requests.get(url)
        data = response.json()

        print(data)
        ```
    </>
